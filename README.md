# CS-405 Secure Software Portfolio (C++)

This repository contains secure software engineering portfolio artifacts completed for **CS-405: Secure Coding** at Southern New Hampshire University. The portfolio demonstrates secure coding concepts in C++ through small focused modules, supporting documentation, testing artifacts, and security policy work.

## Portfolio Contents

| Folder | Purpose |
|---|---|
| `Static-Code-Analysis/` | Static analysis report artifacts and supporting evidence. |
| `Encryption/` | C++ encryption example with input/output data files. |
| `Unit-Testing/` | Visual Studio C++ unit testing project files and test source. |
| `Exception-Handling/` | C++ exception handling example focused on safe program behavior. |
| `Memory-Safety/` | Buffer overflow and memory safety example files. |
| `Injection-Prevention/` | SQL injection prevention example using C++ and SQLite files. |
| `Security-Policy/` | Secure software policy documentation. |
| `Reflection/` | Portfolio reflection document. |

## Security Topics Demonstrated

- Secure coding standards
- Input validation
- Exception handling
- Memory safety and buffer overflow prevention
- SQL injection prevention
- Static code analysis
- Unit testing for secure behavior
- Secure software development lifecycle concepts
- Security policy and governance

## Repository Cleanup Notes

This repository was cleaned for GitHub readability and cloneability. The original submission archives, Visual Studio cache folders, compiled binaries, debug files, and local IDE metadata were removed. Source code, project files, reports, and portfolio documentation were kept.

One large PowerPoint file from the original export was intentionally left out of this cleaned GitHub-ready version because it was approximately 94 MB and would make the repository unnecessarily heavy to clone. Keep that file separately or upload it as a GitHub Release asset if needed.

## How to Use

Clone the repository and open the relevant module folder. Most C++ files can be reviewed directly. Visual Studio project files are included where available.

```bash
git clone https://github.com/rypeguero/Secure-Software-Portfolio-CPP.git
```
